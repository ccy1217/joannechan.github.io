Video game - asteroid game

Overview

This asteroid game is a classic space shooter where players pilot a spaceship through an asteroid field. Dodge and destroy incoming asteroids while trying to survive as long as possible and rack up points. The game saves each player's record, and the leaderboard highlights the top 10 players, adding a competitive edge and motivating players to beat each other’s scores. With simple controls and arcade-style gameplay, it’s easy to pick up and fun to replay.

Methdology

Key Features

  • Scrolling background – immersive space environment that moves as you play.
  • Leaderboard (JSON-based) – records scores and showcases the top 10 players.
  • Asteroid splitting – large asteroids break into two smaller rocks when destroyed, increasing chaos and strategy.
  • Varied bullets – collect items to change your spaceship’s projectile type.
  • Freeze timer – grab the timer item to freeze all asteroids temporarily.
  • Diamond power-up – changes your bullets for a powerful advantage.
  • Coin bonus – collect coins to gain an extra life.
  • Blackhole trap – a dangerous item that reduces two lives if collected.
  • Polished player feedback – responsive sounds and animations that react to gameplay events, making every hit and action more satisfying.

Pattern 1: Finite State Machine (FSM)

Feature Spaceship lifecycle control ( Alive -> Invulnerable -> Dead -> Game Over)

The spaceship transitions between states like:

  • Alive and active
  • Invulnerable after respawn
  • Destroyed and awaiting respawn
  • Game Over

    • Pattern 2: Command Pattern

  • In the game, the Command Pattern is used to handle the spaceship’s bullet-firing mechanism, which switches between a normal bullet and a special bullet based on player interactions (when player collects a diamond).

  • This pattern encapsulates requests (like “fire normal bullet” or “fire special bullet”) as objects, allowing them to be executed dynamically depending on the context.
